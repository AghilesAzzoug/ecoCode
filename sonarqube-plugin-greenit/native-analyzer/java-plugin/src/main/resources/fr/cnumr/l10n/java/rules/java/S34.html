Avoid the use of try-catch-finally when not needed

Noncompliant Code Example

        try {
            int i = 0;
            System.out.println((char) i); //println() doesn't send an Exception
            }
        } catch (IOException e) {
            ....... //never reach
        } finally {
            ...... //never reach
        }

Compliant Solution

        int i = 0;
        System.out.println((char) i);
